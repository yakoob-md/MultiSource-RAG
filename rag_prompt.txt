8. Define function: load_model_int4() -> tuple[processor, model]
   - Create BitsAndBytesConfig(load_in_4bit=True, bnb_4bit_compute_dtype=torch.float16, bnb_4bit_quant_type="nf4")

   - Return (processor, model)

9. Define function: caption_single_image(image_path: str, processor, model) -> str
   - Load image: Image.open(image_path).convert("RGB")
   - Build prompt: "[INST] <image>\nDescribe this image in detail. Include: what objects or people are present, any text visible, the overall context, and what question this image would help answer. Be specific and comprehensive. [/INST]"
   - Tokenize: inputs = processor(prompt, image, return_tensors="pt").to("cuda")
   - Generate: output = model.generate(**inputs, max_new_tokens=400, do_sample=False)
   - Decode: full_text = processor.decode(output[0], skip_special_tokens=True)
   - Extract only the part after [/INST]: return full_text.split("[/INST]")[-1].strip()
   - Wrap entire function in try/except, return "Caption generation failed." on any error

10. Define function: run_caption_pipeline()
    - Print "[Captioner] Starting image captioning pipeline"
    - jobs = get_pending_image_jobs()
    - Print f"[Captioner] Found {len(jobs)} pending jobs"
    - If no jobs: print "No jobs. Exiting." and return
    - Load model: processor, model = load_model_int4()
    - For each job:
      * Print f"[Captioner] Processing: {job['image_path']}"
      * caption = caption_single_image(job['image_path'], processor, model)
      * mark_job_completed(job['id'], caption)
      * torch.cuda.empty_cache()
      * gc.collect()
    - After ALL jobs: del model; del processor; torch.cuda.empty_cache(); gc.collect()
    - Print "[Captioner] Done. Model unloaded from VRAM."

11. At bottom of file:
    if __name__ == "__main__":
        run_caption_pipeline()
